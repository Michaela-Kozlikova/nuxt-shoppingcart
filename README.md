# 🛒 Cyberpunk Shopping Cart (Nuxt Edition)

Rewrite of the Cyberpunk Shopping Cart using the **Nuxt** framework. This project demonstrates the transition from Vanilla JS to a modern, component-based architecture.

## 🛠️ Built With
- **Nuxt 3** (Vue.js Framework)
- **Custom CSS3** (Neon UI & Futuristic Styling)
- **Modern JavaScript**
